# Cytube DOM Map — Navbar

## Main containers

- `nav.navbar` — top navbar wrapper
- `#nav-collapsible` — collapsible Bootstrap nav container

## Useful anchors

- `.nav.navbar-nav` — left-side nav list to append items
- `#welcome` — greeting label
- `#logoutform` — logout form
- `#logout` — logout submit

## Toggles / features

- `#showchansettings` — Channel Settings
- `#layout-nav-toggle` — Layout dropdown menu container
- `#toggle-chat` — “Remove Chat”
- `#remove-video-until-next` — temporary remove video
- `#audio-only` — audio-only
- `#togglemotd` — MOTD toggle
